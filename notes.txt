Vue.js añade data binding y reactividad a nuestras aplicaciones. Ambos conceptos son super importantes si quieres entender y aprovechar todas las ventajas que ofrece una librería tan elegante y minimalista como Vue.

Directives: special token in the markup that tells the library to do something to a DOM element
v-if
v-model
v-for
v-text

new Vue (model view view model) crea una instancia y asocia todas las propiedades del modelo al DOM


Computed properties
Manipulación el modelo antes de renderizar el valor canónico.
Diferencia entre la propiedad computada y los méodos es que los métodos se evalua constantemente y una computed properti solo se evalua cuando cambia la dependencia, en este caso el mensaje, las tareas...En computed properties es una versión cacheada.
